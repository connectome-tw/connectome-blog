Boot Camp

2nd Boot Camp 專訪─抗腫瘤血管新生之標靶藥物傳輸系統

2014年1月19日 Boot Camp 製藥 研究開發 台灣 團隊 新創
過去在癌症治療的研究上，強調找出癌細胞與正常細胞之間的關鍵性差異，並運用這些不同之處給予癌細胞致命一擊，達到治療的效果，此即所謂的「標靶治療」。隨著抗癌標靶藥物不斷推陳出新，化學小分子藥物或蛋白質藥物仍有許多需要克服之處。來自中研院吳漢忠老師實驗室的團隊研發出一套新興的標靶治療模式，這也是未來對於癌症治療與臨床應用的新思維。

2nd Boot Camp 專訪─擬開發山藥萃取物成為美國 FDA 定義之醫療食品

2013年11月15日 Boot Camp 食品 研究開發 台灣 團隊 新創
傳統的中藥遇上新穎的科技，會產生什麼樣的火花？中草藥不管對於疾病或強身都有諸多助益的經驗，用於開發新藥或是發展保健食品也是一種中草藥現在化之選擇。除此之外，還有其他的路嗎？來自中研院楊寧蓀老師實驗室的團隊正開啟了一條與眾不同的道路，也許這就是發揚中國人古老智慧的好方法。

Boot camp 專訪─快速偵測病原的醣晶片

2013年9月3日 Boot Camp 醫材 研究開發 台灣 團隊 新創
「台灣要發展生技，不必像挖金礦那樣，執意一直朝研發新成果投注能量，其實可以從周邊做起，協助別人完成找尋金礦而從周邊獲得利益。」 分別來自生物、化學、化工等不同背景的四位大男生，努力將既有的研究成果整合、發揚光大，藉由參加 Boot Camp 的機會，大大的改變了自己對學術研究到商化應用的想像。

Boot Camp 專訪─Advanced Biotech for Better Cut Flower

2013年8月1日 Boot Camp 畜產、農業 研究開發 台灣 團隊 新創
花卉作物，是台灣重要農業產品之一，當中以切花類（Cut Flower）產值佔最大宗（45%），2011年其產值達新台幣70億元。2012年，中研院育成中心Boot Camp招募之初，對生技創業有高度熱忱的蔡尚燐，集結身旁的英雄好漢組隊參加，以過去實驗室的專利抗菌基因為技術，看準台灣農業生技的優勢，將之運用在切花產業上。他們以技術商品化的模式，模擬狀況並嘗試解決現有市場的困境與需求。Connectome採訪小組深深為Group Ardise的故事所著迷。以下是Group Ardise從招兵買馬、一路摸索修正、到最後找到 Group Ardise價值定位的歷程。

Boot Camp 專訪─癌細胞偵測儀的發想與實現

2013年7月22日 Boot Camp 製藥 研究開發 創業投資 台灣 團隊 新創
幾年前，中研院原分所張大釗實驗室團隊原本為了偵測四股 DNA 的結構，發明了一個分子稱作 BMVC。不過隨後，實驗室前輩意外地發現此分子可選擇性地使癌細胞發出螢光而正常細胞不會有此現象，因此有意將此技術商品化，只是當初洽談過程並不順利。去年，當張老師在實驗室會議中提到中研院育成中心舉辦 Boot Camp 技術價值提升訓練營的時候，本來就對創業有著濃厚興趣的育霖、翊庭兩人當然抓住機會，在實驗室組隊參加。他們參與受訓的題目是想要將此技術發展成一個偵測癌細胞的檢驗儀器。然而，初參與 Boot Camp 時，他們的商業模型還很模糊，連實際病人檢體也沒有碰過。

Boot Camp 專訪─A novel cancer immunotherapy

2013年7月8日 Boot Camp 製藥 研究開發 台灣 團隊 新創
自從 2006 年特殊的 IKDC (interferon-producing killer dendritic cells) 被發現以來，廖南詩老師實驗室便一直不斷投注心力研究這種免疫細胞，此後因為發現 IKDC 同時具有殺死腫瘤細胞，以及誘發 T 細胞免疫反應的抗癌活性，因此開始研究如何將 IKDC 發展成為一種新的癌症療法 - 免疫療法。 2011 年底，一封 Boot Camp 活動的邀請信開啟了廖南詩實驗室團隊的創業夢。負責 IKDC 實驗的爾博、衿菱和彥錰在廖老師的鼓勵和推薦之下，抱著好奇和學習的心，欣然決定參加這項挑戰，從此也開創了人生另一條可能的道路。

Boot Camp 專訪─治療地中海型貧血及鐮刀型貧血症的新藥開發

2013年6月11日 Boot Camp 製藥 研究開發 台灣 團隊 新創
早在三年前，中研院分生所沈哲鯤老師的實驗室，就已經開始著手建立藥物篩選平台，而在Boot Camp 動正式創立之前，實驗室的成員也已經開始利用該平台篩選藥物。過去主修微生物及免疫的周祐吉，在團隊內負責上游藥物篩選模試平台的建立及修正；自生科系畢業的賴正昇，負責藥物篩選後，藥物結構的修飾；至於擅長醫技及遺傳學的陳芮琳，則是負責下游血液相關的臨床研究及動物測試，團隊內依成員所長分工，從上到下各司其職。

Boot Camp 專訪─國產入侵紅火蟻偵測犬

2013年5月29日 Boot Camp 畜產、農業 研究開發 創業家 台灣 團隊 新創
來自中南美洲的紅火蟻每年為台灣帶來上億的損失，也使田野間的戶外活動蒙上了一層陰影。為此，中央研究院在前院長李遠哲博士的指示下，成立了數個紅火蟻研究計畫，希望能夠充分了解紅火蟻的生態、習性以擬出防治對策；中研院物理所陳洋元老師的研究團隊即是最早發展系統性紅火蟻防治方法的團隊之一。 為了進一步探求紅火蟻防治方法商業化的可能性，實驗室中的暉閔、怡臻和琬婷挾著實驗室已發展出的研究成果，在陳老師的支持下於2011年主動參加中研院育成中心所舉辦的Boot Camp活動。藉由一連串技術商業化的訓練，將實際可行的紅火蟻防治方法，推展為可以獨立營運的商業模式，逐步剷除台灣田野、運動場、機場和公園中的紅火蟻巢穴。
