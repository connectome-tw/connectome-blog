台灣的原料藥廠具有一定的競爭力，在國際競爭合作與專利藥將陸續到期的時空背景下，被視為極具有發展潛力的生技領域。Connectome 在此為讀者收集了五家原料藥廠企業名錄，並附上求職連結與介紹，實際幫助讀者了解有哪些企業可供選擇，若您的公司亦屬於此一範疇，歡迎來信告訴我們。

- **台灣神隆股份有限公司** [官網](http://www.scinopharm.com.tw/) [職缺](http://www.104.com.tw/jobbank/custjob/index.php?r=cust&j=3c60436c346c3e6848423a1d1d1d1d5f2443a363189j99)  
  台灣神隆(ScinoPharm Taiwan)為國際性原料藥公司，提供完整的原料藥(API)及中間體的製程研究開發與生產製造服務，擁有專業的技術團隊，針對學名藥及新藥開發客戶供應高價值、高品質且不抵觸專利之原料藥，也提供專利藥廠具競爭效益之商業量產服務。

- **中化合成生技股份有限公司** (中化製藥關係企業) [官網](http://www.ccsb.com.tw/index_ch.aspx) [職缺](http://www.104.com.tw/jobbank/custjob/index.php?r=cust&j=383a436b5636402430683a1d1d1d1d5f24437323189j56)  
  為台灣首家原料藥廠，也是台灣最大的原料藥廠之一；中化合成生技主要從事原料藥的研發、生產與行銷，截至目前為止已開發三十多種原料藥；此外，中化合成生技也提供委託研發與製造的服務。

- **台耀化學股份有限公司** [官網](http://www.formosalab.com/) [職缺](http://www.104.com.tw/jobbank/custjob/index.php?r=cust&j=406042263c5c3e2548423c1d1d1d1d5f2443a363189j48)  
  於2000年踏入原料藥生產的領域。挾其強大的研發實力及厚實的品質系統，逐漸在生技製藥業展露頭角。目前已有多項原料藥及防曬系列活性成份產品行銷全球，而防曬系列活性成分產品為全球前三大供應商。

- **旭富製藥科技股份有限公司** [官網](http://www.sci-pharmtech.com.tw/) [職缺](http://www.1111.com.tw/)  
  本公司為擁有逾百年歷史之瑞士 Siegfried 集團於1987年在台灣所創設，從事中間體、原料藥及特用化學品之研發、生產及行銷，產品外銷歐、美、日醫藥大廠，品質深受客戶肯定。本公司主要股東為三商行集團及瑞士 Siegfried 集團。

- **永日化學工業股份有限公司** (永信集團企業) [官網](http://www.yungzip.com/chinese/0_index/enter.html) [職缺](http://www.1111.com.tw/)  
  以生產人用、藥用醫藥原料及中間體為主。公司係以原料藥之學名藥為研發產品主軸，發展領域包含抗血栓、抗病毒、抗氧化、抗精神病等原料藥。
