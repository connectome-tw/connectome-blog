## **不想走學術研究？- 你怎麼嘗試改變？**

Boot Camp 是 ASCENT 舉辦的活動之一。Boot Camp 的緣起要先談 ASCENT 的創立，至於 ASCENT 創立背後的理念，一定要從創辦人 David Ma 的求學階段說起。

當初在聖路易華盛頓大學（以下稱華大）博二的時候意識到自己未來不想走學術研究。就開始積極尋找方法，要訓練自己如何不要成為一個在象牙塔裡的研究者。很幸運地，當時一個新辦的跨領域課程吸引了 David 的注意。那是一個由華大校長策劃，結合醫學、法、商、理學等學院的 Bio-Entrepreneurship Course。課程的所有內容不論是關於創業訓練、法律、商務、科學[臨床試驗](/job_function/臨床試驗/)等皆是以生物科技為出發點。此課程也鼓勵不同學院的學生來加入這個課程。更棒的是，課程請到的講師皆是擁有生技產業實戰經驗的前輩。學生們找不同背景的同學組成團隊，在學期間要將想法寫成商業提案與營運計畫書（Business Proposal and Business Plan），學期期末報告就是要針對當地的創投來做一個假想的募資簡報。

## <span style="color:#000080">**創立生技創業協會的實作經驗埋下人才培育的種籽**</span>

課程結束後，學生們已經學到理論與一些簡單的實務練習。當時心想：「如果這樣的練習只止於課程，實在是太可惜了！」當初 David 就是以這樣的心情與一群同學一同在華大成立了生技創業協會 [BioEntrepreneurship Core](https://gradpages.wustl.edu/bec)，希望藉由這個社團，讓學生、教授與產業界有更實際的連結機會，培養進入產業的能力。經過四年的努力，BioEntrepreneurship Core 擁有了不錯的規模，而對於實際運作社團的 David 而言，這四年也是建立產業訓練課程的模擬練習。

<span style="color:#800000">最好的學習就是讓同學嘗試失敗</span>，或是有個模擬的機會。其實，這並非陌生的概念，「大家考高中大學前都有經歷無數次的模擬考試，不就是這樣模擬演練的概念嗎？」David 的華大經驗在心裡種下一個培訓學生實作的種籽。

<span style="color:#ffffff">.</span>

## <span style="color:#000080">**改變的力量 - 中研院的跨領域、創業文化引入**</span>

2010年初回台灣任職於中研院育成中心的時候，單位裡沒有任何的培訓課程。David想來只覺得可惜，在中研院育成中心這樣一個這麼好的產學介面，除了做技術的商業化評估，David 心中還想做的是人才培育。這個想法在醞釀了一年後越來越完整，終於，2011 年的時候，David 結合中研院育成中心和技轉辦公室，創立了 Academia Sinica Creative Entrepreneurship Network and Tech Transfer Program（簡稱 [ASCENT Program](https://www.facebook.com/pages/ASCENT/480105248690252?fref=ts)），目的是把跨領域的文化帶進中研院。ASCENT 主要活動包含每月舉辦的系列演講 (Seminar Series)、一年一屆歷時五個月的智財讀書會（IP Study Group），另外就是 Project-based 的技術商品化培訓營（Boot Camp）。他們希望透過創業文化<span style="color:#808080">註</span>塑成，開始養成年輕人有正確創業及就業的態度。

<span style="color:#808080">註：此創業文化塑成並非鼓勵所有人都去創業，此創業文化指的是敢於創新的勇氣、科技與商業的眼光、兢兢業業的態度的綜合。如同運動家精神，每個人都可以有運動家精神，但並非每個人都是專業運動員。</span>

<span style="color:#ffffff">.</span>

## <span style="color:#000080">**ASCENT Boot Camp - 創辦初期的挑戰**</span>

Boot Camp 策劃過程，雖然經費少，但是 David 感激梁啟銘主任給予極高的自由度，讓他可以無後顧之憂的去嘗試與突破。過程當中還是有面臨一些挑戰，這些反對的聲音主張中研院是一個基礎研究單位，不應舉辦這樣「不務正業」的課程。但是 David 心意堅定，他以在美國看到的經歷成功說服反對者。在美國與中研院差不多等級的研究機構內或是相關的支援單位都有提倡或是執行推動 Entrepreneurship 的單位，在中研院這樣一個人才集結的地方更應該要有系統性的人才培育計畫。世界的門已經打開，台灣最高研究單位沒有理由把機會之門關起來。

Boot Camp 最大的賣點其實是一個簡單的道理：「請職棒球員來教想打棒球的人如何打棒球！」David 笑著給了我們這樣淺顯易懂的比方，「如果我們想打好棒球，我們不會找球評或甚至是籃球球員、足球球員來教我們打棒球嘛！」David 延續在華大的精神，找來生技業創投、創業家、律師、等等實際在戰場上的前輩業師，提供參與者第一手的「戰情分析」。

第一屆 Boot Camp 是 2011 年底開始舉行。儘管請來的業師陣容堅強，最初報名的情況不甚理想。David 於是從[技轉](/industry/育成/技轉/)中心研究有潛力的實驗室主動聯繫，同時也整理宣傳過程中每封 PI 來信提出的任何疑問。他一間間拜訪解釋，宣傳此活動。他們發現大部份老師擔心的是技術的專利安全問題。David 解釋，事實上，Boot Camp 會確定報名者提出的 project 在專利上是有受到保護的才會接受報名。最適合的 project 就是有潛力商品化、已有專利保護但是目前不是實驗室最前端技術者。「對一個訓練課程而言，project 所使用的技術最終是否可以成為商品並不是重點，而是訓練所提供的學習經驗。」所以 David 說，「參與者甚至可以拿一個實驗室早已開發許久的 A 技術來參加 Boot Camp，活動結束後，參與者就有能力利用所學的經驗來開發實驗室裡確實有商品化潛力的 B 技術、C 技術。」

## <span style="color:#000080">**ASCENT Boot Camp 如何運作？又給學員帶來些什麼？**</span>

經過一番苦心，第一屆 Boot Camp 有 14 隊伍報名參加、其中 12 隊完成了包含技術價值提升訓練活動、每月一次的互動式追蹤聚會以及最後的訓練成果分享，為期半年以上的計劃。過程當中，他們除了 Science，法規、市場、[專利](/job_function/智財管理/)如何佈局要學、更要學會[跨領域](/topic/學習與跨領域/)的團隊合作與如何跟潛在投資人溝通。這些隊伍從實作中學習，Boot Camp 就是最好的模擬創業環境。<span style="color:#800000">這種 project-based 訓練之可貴即在於，幾個月下來，業師量身指導修改題目的務實性，給予市場上第一手的建議</span>。

Boot Camp 的培訓目標並非期待立即有創業隊伍出現。「人才培育是百年樹人的概念，不可能立竿見影、並不是說你計算投入原料的成本，就能預估立即的量化成果出來。我們不能用這樣工業代工有短期成效的思維來看待人才培育。」David 眼神中充滿力量地說，「<span style="color:#800000">我們做 Boot Camp 希望可以促成一些創業的能量，這些受訓的老師同學，往後看研究題目可以用不同於別人的角度</span>。」

<span style="color:#ffffff">.</span>

## <span style="color:#000080">**ASCENT Boot Camp 的現況與未來**</span>

現在 Boot Camp 已經進行到第二屆了。第二屆負責人蔡尚燐（Shawn）表示，有了 David 第一屆的披荊斬棘，第二屆在招募和宣傳上順利許多，整個活動框架與工作團隊漸趨完備；第一屆參與的團隊多為與[醫材](/industry/醫材/)相關；而第二屆則有許多新藥開發相關團隊加入。雖然，截至目前為止已有許多團隊因臺灣法規限制或技術開發進度受限而止步，以主辦單位的角度而言，無法完成訓練十分可惜。不過，或許也就因為這樣的經驗，讓參與者能夠深刻體會到技術商品化的過程中每個環節都有其重要性存在。ASCENT 的終極規劃，乃是計劃以出國比賽的形式將 Boot Camp 團隊推向國際舞台，讓團隊成員能有效地與國際接軌，從大輸大贏中定位自我、出類拔萃、培養格局。然而，目前育成中心尚無如此龐大資源與資金，但是仍會持續努力讓這樣的概念得以在未來實現。

<span style="color:#ffffff">.</span>

## <span style="color:#000080">**台灣生技產業的未來展望 - Are You Prepared?**</span>

David 說，<span style="color:#800000">生技人才若要在全球競爭中突出，需要廣泛且扎實的科學底子、某項特別專精的技術或者知識、優秀的溝通能力還有創業家精神。</span>而 Boot Camp 或許就是協助生技人習得溝通能力與創業家精神的重要平台。David 也提到，作為移民國家，台灣人性格原本就充滿著創業家精神，但隨著時間的演進，青年們卻漸漸習慣「被動成長」，造成產業上創新與創業的動能日漸衰弱，十分可惜。也就是如此，我們能夠發現，ASCENT 的每一步，都是一個盼望，盼望台灣創業家精神的復興，盼望台灣生技產業能夠藉由充實的準備與勇氣，在將來得到成就與自信。<span style="color:#ffffff">.</span>
