# NTU Prestige Project II

R&D：研發部門藥做什麼?  
全國的生技製藥產業正逐漸乘著人的夢想而起飛  
灑下希望的種子在世界各國萌芽  
台灣的嫩芽也在這一波洗禮下茁壯  
而在研發門檻越來越高的潮流中  
研發新藥對於剛發芽的台灣  
是個神話? 抑或是個契機?

這次Prestige請到東洋製劑研發中心副總 胡宇方 博士  
來介紹台灣藥業的R&D趨勢及經驗分享  
帶大家認識研發部門的工作  
如何準備自己、掌握機會  
如此難得不好好把握怎麼知道  
要如何抵抗這股生技製藥圈的狂風巨浪?

講師：胡宇方 博士  
簡歷：  
TTY台灣東洋製劑研發中心 副總經理  
Diatide Inc. NH USA  專任SR.Chemist  
Immulogic Pharmaceutical USA Scientist  
St. John's University 藥劑博士
