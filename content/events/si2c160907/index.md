# 「製造於以色列」的台灣創業家:

# 與伊勒伯創辦人兼執行長熊樂昌博士的對談

**About**

以色列面積只有台灣的三分之二，卻擁有全世界第二大的創新生態系統，人均創業密度為全球第一，其中第二大城市特拉維夫(Tel Aviv)更是創業聖地，被稱為矽谷第二。以色列是如何創造出如此蓬勃發展的創業生態系統? 與以色列同樣擁有不大的內需市場、出口導向、本土資金規模小但技術人才充足等特性的台灣，是否可以從中得到借鏡? Si2C SPARK Taiwan這次邀請到”Made in Israel”的台灣創業家熊樂昌博士，分享他在以色列學習及創業的經驗，如何透過成功的技術商品化經驗，獲得國際資金挹注並在台灣打造「以色列風格」的新創公司-伊勒伯科技(Winnoz)。熊博士並將分享以色列新創生態環境以及政策制度、以色列生技醫藥產業概況、當地社群連結方式及強度、以及投資環境、投資人樣態及投資觀察重點等。台灣及以色列能否憑藉各自的強項及互補之處，協同兩國的優勢打造完美的合作模式? 熊博士此次更將以自身創業經驗，鼓勵台灣年輕人「創新思維、適當規畫、勇於冒險、充分溝通、起身執行接受並學會被否定」!

**Speakers**

**熊樂昌 博士**

熊樂昌博士為伊勒伯科技股份有限公司(Winnoz)創辦人兼執行長。熊博士曾於以色列特拉維夫大學(Tel Aviv University)進行三年博士後研究，致力於體外診斷裝置的研發及商品化。2014年回台後與台灣科技大學機械系教授合作成立公司，團隊成員來自不同國家並包含大學教授、化學家、醫生、工程師等專家，並成功吸引國外資金投資，今年四月更於以色列成立分公司。熊博士帶領Winnoz從台灣連接到以色列，致力將台灣的創新檢測技術帶到匯聚眾多國際買家的以色列。
