# NTU Prestige project I    醫藥代表價值、任務與客戶關係

[課程內容]

醫藥代表的角色是───  
代表公司為客戶做產品銷售服務、  
傳達產品處方資訊給醫藥單位、  
將客戶對產品反應意見回饋給公司。

而您對醫藥代表的任務與價值之認知為何呢？  
藥廠對於業務代表經常只著重於工作能力的培養，但其精神使命與價值值得各位細細思考，本次課程講師融合實做活動與引導思考：  
在提升業績同時，是不是有什麼方法可以讓自己跳脫社會對業務的刻板印象？  
一位優秀的醫藥代表如何建立自己以及產品的價值？  
如何保持並運用自己的專業讓自己變的更出色、更有競爭力？

繼三堂精彩的Sales實務課程──  
醫藥品銷售術實務研習、專業藥品銷售技巧實戰模擬、藥商行銷大剖析，  
有幸再次邀請 賴俊良藥師為Prestige投下第一彈課程！  
讓我們與有豐富醫藥代表之經歷的 賴俊良藥師，一同深究醫藥代表甚少提及的層面-如何更能凸顯醫藥代表的價值！
